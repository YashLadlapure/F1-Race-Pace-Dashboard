# Tableau Dashboard

**Live link:** https://public.tableau.com/app/profile/yash.ladlapure/viz/F1_Race_Pace_Dashboard_2010_2026/Dashboard1

**Canvas size:** 1200 x 800 px (fixed)
**Layout:** Tiled 2x2 grid
**Background:** Dark navy
**Title font:** White (#FFFFFF), Bold

## Sheets

**Season Pace** — Line chart. Race Year on columns, AVG(Avg Lap Sec) on rows. Color: coral.

**Top 15 Drivers** — Horizontal bar chart. Driver Id on rows, AVG(Avg Lap Sec) on columns. Filtered to top 15 by measure value, sorted ascending.

**Pit Strategy** — Dual-axis line chart. Race Year on columns, AVG(Avg Lap Sec) on first axis, AVG(Pit Pct) on second axis. Axes synchronized.

**Best Lap Per Season** — Scatter/dot chart. Race Year on columns, AVG(Best Lap Sec) on rows. Mark type set to Circle.

## Dashboard elements

Title text box at top. Two KPI text boxes (91.2 Avg Lap Time, 372,493 Total Laps). Subtitle bar with project info. Four tiled chart containers in 2x2 layout.
